# CRO Qualification Framework

## ICP primario — Property Management

- **Tamaño:** 500–5.000 unidades bajo gestión.
- **Employees:** 15–100.
- **Geo:** US (primary), LatAm y Europa (secondary).
- **Dolores clave:** lead response lento, lease renewals manuales, mantenimiento caótico, reporting fragmentado.
- **Perfil de sponsor:** Director of Operations, Controller, CFO, COO.

## ICP secundarios (no abrir hasta tener 3 clientes PM)

- Accounting/bookkeeping firms: 10–80 contadores. Dolor: data entry, reconciliación manual.
- Legal boutique: 5–40 abogados. Dolor: intake lento, research manual.
- B2B SaaS mid-market: 20–200 empleados. Dolor: prospección lenta, CRM manual.

## Criterios de calificación — pasar a "audit proposed"

El lead debe cumplir ≥ 3 de los 5:

- [ ] Dolor claro y económicamente cuantificable.
- [ ] Sponsor ejecutivo en la llamada o identificable.
- [ ] Budget > 5K USD disponible.
- [ ] Timing dentro de 90 días.
- [ ] Stack tecnológico razonable (no legacy bloqueante).

Si < 3 → marcar "needs nurturing" → secuencia de touchpoints cada 30 días.

## Stage definitions

| Stage | Descripción | Probabilidad |
|---|---|---|
| New | Lead entró, no contactado | 5% |
| Engaged | Reply positivo o interacción | 15% |
| Qualified | Cumple ≥ 3 criterios ICP | 35% |
| Discovery | Discovery call hecha, dolor confirmado | 50% |
| Audit Proposed | Propuesta enviada | 65% |
| Audit Sold | Contrato firmado, pago recibido | 100% |
| Nurturing | Interés pero timing no ahora | 10% |
| Closed Lost | No continúa | 0% |

## ICP Score (0–100)

| Señal | Puntos |
|---|---|
| Vertical exacto (property mgmt) | +25 |
| Tamaño correcto (500-5K unidades) | +20 |
| Rol correcto (CFO/COO/Controller/Dir Ops) | +15 |
| Pain statement mencionado explícitamente | +20 |
| Budget señalado (> $5K) | +10 |
| Timing < 90 días | +10 |

Score ≥ 70 → prioridad alta, outreach personalizado inmediato.
Score 40–69 → secuencia estándar.
Score < 40 → nurturing largo o descartar.
