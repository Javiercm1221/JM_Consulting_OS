# JM Consulting — Knowledge Base

Core knowledge files for all JM Consulting agents. Contains company identity, operating model, pricing, org structure, policies, and glossary.

## Contents

- **mission_vision.md** — Mission, vision, positioning, and current state (April 2026).
- **core_values.md** — 8 operating values and anti-patterns.
- **org_chart.md** — 6-agent org structure, heartbeat schedules, approval matrix.
- **pricing_service_ladder.md** — 5-tier service ladder with full pricing, ROI formula, discount rules.
- **heartbeat-policy.md** — When and how agents run heartbeats (rhythmic, event-driven, on-call).
- **approvals-policy.md** — Approval levels (0/1/2), SLAs, escalation rules.
- **glossary.md** — 30+ defined terms: business, audit, operational, outbound.
- **client_conduct.md** — Client relationship principles, scope policy, payment policy, communication model.

## Usage

All agents should load this skill to have full context on JM Consulting's identity, services, and operating rules before responding to any task.
